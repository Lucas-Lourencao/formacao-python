Instalando e trabalhando com Ambiente Virtual no VSCode:

1) Criar a pasta do projeto;
2) Dentro da pasta do projeto abrir o prompt e dar o comando:
>>> python -m venv venv

3) Ativar o ambiente virtual, prompt cmd:
venv/Scripts/activate.bat

4) Selecionar o caminho do ambiente virtual, o que automaticamente criar√° a pasta .vscode;

5) configurar o settings.json para o trabalho;

6) Para desativar o venv, prompt de cmd:
deactivate
